🚀 DEPLOYMENT STEPS:

1️⃣  TRANSFER TO YOUR DROPLET

   Option A - Direct SCP (if you have SSH key):
   ────────────────────────────────────────────────
   scp -r deploy-package root@YOUR_DROPLET_IP:/tmp/quantum-nexus


   Option B - Download from Replit:
   ────────────────────────────────────────────────
   1. In Replit, click "Files" → "Export as ZIP"
   2. Download and extract the zip file
   3. Upload deploy-package/ to your droplet:
      scp -r deploy-package root@YOUR_DROPLET_IP:/tmp/quantum-nexus


2️⃣  RUN THE INSTALLER

   SSH into your droplet:
   ────────────────────────────────────────────────
   ssh root@YOUR_DROPLET_IP

   Navigate and run installer:
   ────────────────────────────────────────────────
   cd /tmp/quantum-nexus
   chmod +x install-on-droplet.sh
   ./install-on-droplet.sh

   ⏱️  Installation takes 5-10 minutes


3️⃣  VERIFY DEPLOYMENT

   Check service status:
   ────────────────────────────────────────────────
   sudo systemctl status quantum-nexus

   Test API health:
   ────────────────────────────────────────────────
   curl http://localhost:8000/healthz

   Access your app:
   ────────────────────────────────────────────────
   http://YOUR_DROPLET_IP


4️⃣  SETUP SSL (PRODUCTION READY)

   Install Certbot:
   ────────────────────────────────────────────────
   sudo apt install certbot python3-certbot-nginx -y

   Get certificate (replace with your domain):
   ────────────────────────────────────────────────
   sudo certbot --nginx -d yourdomain.com

   Enable firewall:
   ────────────────────────────────────────────────
   sudo ufw allow 'Nginx Full'
   sudo ufw enable

❓ TROUBLESHOOTING:

  Service won't start:
    → Check logs: sudo journalctl -u quantum-nexus -xe
    → Verify .env file: ls -la /var/www/quantum-nexus/.env
    → Test manually: cd /var/www/quantum-nexus && source venv/bin/activate && python -m app.main

  Database connection failed:
    → Check PostgreSQL: sudo systemctl status postgresql
    → Verify database exists: sudo -u postgres psql -l
    → Check credentials in .env file

  Nginx 502 error:
    → Backend not running: sudo systemctl start quantum-nexus
    → Check backend port: netstat -tlnp | grep 8000

  Can't access from browser:
    → Check firewall: sudo ufw status
    → Verify Nginx: sudo systemctl status nginx
    → Check server IP: curl ifconfig.me
